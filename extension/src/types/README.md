# `src/types/`

Shared TypeScript contracts for bridge/runtime/storage.

## Included Models

1. bridge message schema:
   1. `BridgeRequestMessage`
   2. `BridgeResponseMessage`
2. WebAuthn payload shapes:
   1. creation (`create`)
   2. assertion (`get`)
3. storage domain models:
   1. `PersistedCredential`
   2. `Settings`
4. algorithm domain:
   1. `SupportedAlg` (`-48/-49/-50`)
